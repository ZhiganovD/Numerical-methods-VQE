6.- CREATING GIT CLONE REPOSITORY IN COMPUTER

	- Log in in VsCode (bottom left) with the email of github
	- On left pannel, click: Source Control symbol
	- Click: Clone Repository
	- Paste: https://github.com/ZhiganovD/Numerical-methods-VQE.git
	- Choose a folder to store the repository
	

7.- PUSHING LOCAL CHANGES TO THE WEB REPOSITORY
    - FIRST TIME ONLY:
        -   Open Ubuntu
        -   Run: git config --global user.name yan-drak  #change for github username
        -   Run: git config --global user.email gregortobaschus@gmail.com   #change for github email
	
    - Save file in VsCode
	- Click: Source Control symbol, on the left pannel
	- Click: '+' symbol on the right of CHANGES
	- Add a message (eg: "algorithm change") and click: 'Commit and Sync' on the right of 'Commit' ('v' symbol)


8.- DOWNLOADING REPOSITORY CHANGES TO YOUR LOCAL COMPUTER
	- Open VsCode
	- Click: Source Control symbol
	- On the Graph section appearing below the CHANGES section, click: 'Fetch from all remotes' symbol (arrow pointing down symbol, on the right of GRAPH)
	- Click: 'Sync Changes' (in the CHANGES section)
